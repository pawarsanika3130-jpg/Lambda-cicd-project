# 🚀 Automated CI/CD Pipeline using AWS Lambda & CodePipeline

## 📌 Project Overview

This project demonstrates a fully automated CI/CD (Continuous Integration and Continuous Deployment) workflow on AWS.

Whenever code is pushed to GitHub, AWS CodePipeline automatically detects the change, triggers a build process using CodeBuild, and invokes an AWS Lambda function to automate deployment activities. CloudWatch provides centralized monitoring and logging for the entire workflow.

The solution eliminates manual deployment efforts and showcases modern DevOps automation using AWS managed services.

## 📂 Workflow

### Source Stage
- Code pushed to GitHub
- Webhook triggers CodePipeline

### Build Stage
- CodeBuild downloads source code
- Dependencies are installed
- Validation and build processes are executed

### Deploy Stage
- Lambda function is invoked
- Deployment workflow starts automatically

### Monitoring
- CloudWatch captures logs
- Tracks pipeline execution status
- Records deployment events
